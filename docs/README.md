# Yaro0 Documentation

This folder organizes the documentation from `yaaro0_documentation.md` by area.

- `api/` - API endpoint notes and request/response contracts
- `database/` - schema, migrations, seed data, and ERD notes
- `backend/` - Laravel API setup and architecture notes
- `frontend/` - Next.js public site and admin panel notes
- `mobile/` - Flutter app setup and feature notes
- `deployment/` - production server, DNS, queue, websocket, and release notes
